# Student Management System (Java)

This project is a simple Student Management System developed using Java Swing and file handling with crude operations.

## Features
- Add Student
- Search Student
- Update Student
- Delete Student
- Display All Students

## Technologies Used
- Java
- Java Swing
- File Handling

## How to Run

Compile the program:

javac StudentManagementGUI.java

Run the program:

java StudentManagementGUI

## Data Storage
Student records are stored in a file called Student.
